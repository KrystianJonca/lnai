# Installation

## Requirements

- Node.js 22 or later
- pnpm (recommended) or npm

## Install globally

```bash
# Using pnpm
pnpm add -g @lnai/cli

# Using npm
npm install -g @lnai/cli
```

## Install as dev dependency

```bash
# Using pnpm
pnpm add -D @lnai/cli

# Using npm
npm install -D @lnai/cli
```

## Verify installation

```bash
lnai --version
```
